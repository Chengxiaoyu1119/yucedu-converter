# YUCEdu 双向转换器 2.1.0 使用说明

## 第一次使用

1. 启动程序。
2. 选择“解密 YUCEdu”或“加密视频”。
3. 点击“选择文件夹”，决定本次输出位置。
4. 添加文件或文件夹。
5. 点击“开始转换”。

输出位置每次启动后都由用户主动选择。

## Windows 播放

- 普通视频：PotPlayer、VLC 或 Windows 默认播放器。
- `.yucedu`：`WinNetPlayer1018.exe`。

## macOS 播放

- 普通视频：IINA、VLC 或 macOS 默认播放器。
- `.yucedu`：`MacNetPlayer.app`。

播放器路径可在设置窗口中手动选择。

## 支持输入

解密模式接受 `.yucedu`。加密模式接受：MP4、MKV、AVI、MOV、M4V、WMV、FLV、WebM、TS、MPEG 和 MPG。

## 同名文件

- 自动改名：生成 `文件名 (1)`。
- 已有文件时停止：保留已有文件。
- 确认后覆盖：开始任务前再次确认。

## 临时文件

转换时会生成隐藏临时文件。成功后临时文件会替换为正式输出；取消或失败时会自动清理。

## 快捷键

| 功能 | Windows | macOS |
|---|---|---|
| 添加文件 | Ctrl+O | Command+O |
| 添加文件夹 | Ctrl+Shift+O | Command+Shift+O |
| 开始转换 | Ctrl+Enter | Command+Enter |
| 移除所选 | Delete | Delete / Backspace |

## 日志

Windows：

```text
%LOCALAPPDATA%/YUCEdu双向转换器/日志/
```

macOS：

```text
~/Library/Application Support/YUCEdu双向转换器/日志/
```
