# YUCEdu 双向转换器 2.0.1 使用说明

## 操作顺序

```text
选择模式 → 选择输出文件夹 → 添加文件 → 开始处理
```

## 解密

1. 选择“解密 YUCEdu → 视频”。
2. 点击“选择文件夹”，决定本次视频保存位置。
3. 添加一个或多个 `.yucedu` 文件。
4. 点击“开始解密”。
5. 完成后双击任务播放输出视频。

## 加密

1. 选择“加密视频 → YUCEdu”。
2. 点击“选择文件夹”，决定本次文件保存位置。
3. 添加支持的常见视频。
4. 点击“开始加密”。
5. 完成后双击任务，可使用本地 WinNetPlayer1018 验证。

## 输出位置

- 每次启动后输出位置为空。
- 解密与加密模式在当前运行中分别保存所选目录。
- 新一轮启动再次由用户选择。
- 输出位置未选择时，开始按钮保持灰色。

## 文件处理

- 输入文件保持原样。
- 默认自动改名保护已有文件。
- 转换过程先写临时文件，校验成功后提交最终文件。
- 取消任务时清理当前未完成的临时文件。
