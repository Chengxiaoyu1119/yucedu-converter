# 兼容报告

## 转换核心

- 流式加密与解密保持常量内存占用。
- 当前配置资源在运行前执行大小和 SHA256 校验。
- 失败或取消后清理临时文件。
- 加密结果追加当前兼容尾部。
- 解密结果执行媒体类型识别和 MP4 结构检查。

## 播放器调用

- Windows：PotPlayer、VLC、系统默认播放器、WinNetPlayer1018。
- macOS：IINA、VLC、系统默认播放器、MacNetPlayer。

真实播放器与课程媒体保存在仓库外。公开自动测试使用合成数据，本机真实样本通过环境变量参与回归。

## 已知边界

- 当前配置只覆盖已经验证的 `.yucedu` 分支。
- macOS 公开构建采用 ad-hoc 签名，首次启动需要通过 Control 点击选择“打开”。
- macOS 12–14 的最低版本字段已设置为 12.0，正式兼容结论仍以对应系统实机结果为准。
