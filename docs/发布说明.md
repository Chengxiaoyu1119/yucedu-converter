# YUCEdu 双向转换器 2.1.0

本版本新增 macOS Intel 与 Apple Silicon 应用，并把 Windows、macOS 构建统一到自动 Release 流程。

## 下载文件

| 文件 | 适用设备 |
|---|---|
| `yucedu-converter-v2.1.0-windows-x64.zip` | Windows 10/11 x64 |
| `yucedu-converter-v2.1.0-macos-arm64.dmg` | Apple Silicon Mac |
| `yucedu-converter-v2.1.0-macos-x64.dmg` | Intel Mac |

每个安装包旁边都有同名 `.sha256.txt` 校验文件。

## Windows 安装

完整解压 ZIP，然后双击 `YUCEdu双向转换器.exe`。`运行组件` 文件夹需要与 EXE 保持在一起。

## macOS 安装

打开 DMG，把 `YUCEdu双向转换器.app` 拖入 `Applications`。首次启动时按住 Control 点击应用并选择“打开”。

当前 macOS 构建经过双架构 PyInstaller 构建、启动冒烟检查、Info.plist 检查、架构检查和 ad-hoc 签名完整性验证；Apple Developer ID 公证将在后续签名阶段接入。

## 主要变化

- 新增 macOS arm64 与 x64 图形应用。
- 支持 macOS 默认播放器、IINA、VLC 和自定义 `.app`。
- 支持查找并调用 `MacNetPlayer.app`。
- 设置、日志、字体、快捷键和文件管理器调用按平台适配。
- 项目版本改为单一来源。
- Git 标签自动创建包含 Windows 与 macOS 资产的 Release。
- CI 覆盖 Windows Python 3.11/3.13 和 macOS arm64 Python 3.13。

详细操作见 [使用说明](使用说明.md) 和 [macOS 使用说明](macOS说明.md)。
