# 多平台路线图

## 已完成：Windows 与 macOS 参考实现

- Windows 10/11 x64 图形应用。
- macOS Apple Silicon 与 Intel 图形应用。
- Windows ZIP、macOS DMG 和 SHA256 自动发布。
- 跨平台设置、播放器、日志、字体与快捷键适配。
- GitHub Actions 多平台构建与 Release。

## 下一阶段：格式规范与 CLI

- 增加 `inspect`、`encrypt`、`decrypt`、`verify` 命令。
- 建立合成测试向量和固定 SHA256。
- 把格式配置、兼容尾部和算法版本显式化。

## 共享核心评估

根据 CLI 和测试向量的性能结果，再决定 Rust `yucedu-core`：

- 流式加密与解密。
- 媒体类型识别。
- 进度与取消。
- 临时文件和原子提交。
- 配置档案与兼容测试。
- C ABI / FFI 供桌面和移动界面调用。

Python 版本继续作为正确性参考实现。

## 后续平台

- Linux：x86_64、arm64、CLI、AppImage 与桌面 GUI。
- Android：文件导入、批量任务、后台进度与导出。
- iOS / iPadOS：Files 导入、转换、导出和内置播放。

## 长期模块

```text
yucedu-core          跨平台转换核心
yucedu-cli           命令行与自动化
yucedu-desktop       Windows / macOS / Linux
yucedu-mobile        Android / iOS
yucedu-player        流式读取与播放
yucedu-compatibility 固定测试向量和播放器兼容矩阵
```
