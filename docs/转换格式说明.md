# 转换格式说明

## 解密输入

```text
.yucedu
```

当前已验证媒体输出为 MP4。程序会检查媒体头、顶层 MP4 Box、兼容尾部和资源 SHA256。

## 加密输入

```text
.mp4 .mkv .avi .mov .m4v .wmv .flv .webm .ts .mpeg .mpg
```

转换只处理文件字节，不进行视频重新编码，因此画质、分辨率、码率、音轨和字幕流保持原样。

## 可靠性

- 分块处理大文件。
- 写入 `.转换中.tmp` 临时文件。
- 校验成功后提交最终文件。
- 取消或错误时清理本次临时文件。
- 默认自动改名保护已有输出。

## 配置范围

不同来源的 `.yucedu` 可能使用不同转换配置。当前版本明确维护一套经过回归测试的配置，并通过资源长度、SHA256 和固定样本结果进行校验。
