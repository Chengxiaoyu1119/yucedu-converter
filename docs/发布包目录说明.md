# 发布包目录说明

GitHub Release 只保留两个下载文件：

```text
yucedu-converter-v2.0.1-windows-x64.zip
yucedu-converter-v2.0.1-windows-x64.zip.sha256.txt
```

ZIP 解压后只有一个版本目录，内容按用途分组：

```text
yucedu-converter-v2.0.1-windows-x64/
├─ YUCEdu双向转换器.exe       # 双击启动
├─ 运行组件/                  # 程序依赖，请保持原位置
├─ 使用说明.md                # 新手操作说明
├─ SHA256校验值.txt           # 核心文件校验值
└─ 文档/
   ├─ 目录结构.md
   ├─ 版本说明.md
   ├─ MIT许可证.txt
   ├─ 第三方组件说明.md
   └─ 界面预览/
```

## 使用约定

- 日常使用只需双击 `YUCEdu双向转换器.exe`。
- `运行组件` 必须与主程序放在同一目录，不要单独移动或改名。
- 输入视频和输出视频由用户自行选择位置，不写入程序安装目录。
- 升级版本时，完整解压新 ZIP，不与旧版本目录混合覆盖。
