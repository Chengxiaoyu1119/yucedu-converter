# 更新记录

## 2.1.0 - 2026-07-16

### 新增

- macOS Apple Silicon arm64 与 Intel x64 图形应用。
- macOS DMG、SHA256、App 启动和签名完整性检查。
- macOS 默认播放器、IINA、VLC 与 MacNetPlayer 调用。
- GitHub 标签自动创建 Windows 与 macOS 多平台 Release。

### 改进

- 项目版本改为单一源码来源，构建脚本自动读取和校验。
- CI 覆盖 Windows Python 3.11/3.13 与 macOS arm64 Python 3.13。
- Windows 构建增加正式 EXE 启动和任务栏图标验收。
- 设置、日志、字体、快捷键和文件管理器操作按平台适配。

### 发布资产

- Windows x64 ZIP。
- macOS arm64 DMG。
- macOS x64 DMG。
- 三个平台对应的 SHA256 校验文件。

## 2.0.1 - 2026-07-16

### 新增

- 解密 `.yucedu` 为普通视频。
- 把常见视频加密为 `.yucedu`。
- 批量文件、文件夹、进度、取消和自动改名。
- Windows 标题栏和任务栏图标。
- 当前已验证播放器分支的兼容加密。

### 改进

- 输出文件夹改为每次由用户主动选择。
- 任务较少时自动隐藏滚动条。
- 任务较多时显示无箭头的细条式滚动条。
- 输出位置移动到任务列表上方。
- 设置页集中管理播放器、同名文件和技术配置。

### 验证

- 20 项本地自动测试通过。
- 当前回归样本解密后重新加密，与原文件逐字节一致。
- Windows 最小 PATH 环境启动通过。
- ZIP 全文件哈希校验通过。
